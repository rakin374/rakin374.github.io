Introduction

The Berkeley Box is the original name I gave to a collection of software and deep-learning based tools for music. The idea is to just use the most current tools to enhance musical exploration, creation, etc. The project was started as a part of CS505 @ BU. Slowly it turned into a multi modal project, hence the term brain. The first "feature" of this brain is to be able to learn style from any artist and play piano music in that style. It used a diffusion-based deep learning model. The details can be found in the paper linked. Methods used were off-the-shelf and ad-hoc but demonstrated proof of concept. The fundamental token was color values in an RGB image.

organic brain structures

The scope of what a brain "is", is pretty large. When we think of brains we think of mammilian ones. But much smaller creatures such as flies have brain too. They have a central nervous system that controls their behavior. If we think of a brain as a network fungi and plants start to emerge as brain too. The main point here is the difficulty of completely defining a brain.
To take a more pragmatic approach we decide to loosely define brains as those networks found biological assemblies which are the central information processing centers for that assembly. These processing systems can be simple and small with the assembly itself being the network. E.g. a single neuron assembly. Or they can be complex with special mechanics such as the human brain.

artifical brain structures

When I write artifical brain I don't mean a non organic sponge like membrane inside of a special fluid vaccuum inside of some genius biologist's lab. Rather I mean a system made from artifical computational components which provide cognitive function that mimics that of a "natural"/"organic" brain. This litmus test was inspired partially by the Turing Test.
The brains that drive today's ai systems are built using feed forward artifical neural networks. These networks make use of the universal approxmiation theorm and simply learn to optimize against loss. These networks are learned using gradient based algorithms. But what else is there?
